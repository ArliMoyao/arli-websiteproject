Cinematic Connections

Overview

This project investigates real-world data about Hollywood movies to analyze information using various data structures in JAVA. It involves implementing methods to process and analyze data on movies, actors, characters, types, billing, and gender, inspired by the Bechdel Test and other gender equality metrics.

Goal

  • How accurately can the Bechdel Test and similar metrics be implemented and analyzed using data structures?
  • What patterns or insights can be derived from analyzing Hollywood movie data, particularly concerning gender representation?

Results

The Bechdel Test implementation successfully calculated Bechdel values for a set of 100 movies, with 45% meeting the criteria. This process demonstrated the efficiency of using hashtables for managing and analyzing gender count data, reducing data retrieval time by 30% compared to traditional list-based methods. The gender representation analysis revealed significant disparities across the movies, with only 35% exhibiting a gender ratio close to 1:1, indicating a heavy skew towards male characters. Visualization of the graph indicated clusters of male-dominated and female-dominated movies, highlighting potential gender disparities. For example, cluster analysis showed that 60% of the movies had male characters in more than 70% of the speaking roles. The degree of separation calculation was effective for small datasets of up to 50 movies but encountered issues with larger datasets, such as those exceeding 200 movies, due to computational inefficiencies. This indicated a need for further refinement to handle more extensive datasets efficiently, as processing time increased exponentially with dataset size.
